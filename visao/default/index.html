Fácil MVC

Um framework PHP/HTML5 muito, muito fácil

Primeiros passos:

* Edite a classe controlador\Home
- Ela representa o módulo principal (e único já existente) do controlador
- Cada método seu é uma URL possível
* Para despachar uma view utilize Facil::despachar($_view);
- $_view é o nome de um arquivo presente no diretório visao/default
